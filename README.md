# 洛克王国：世界 · 宠物蛋组查询器 & 种公推荐

> 纯前端、离线可用，支持种公匹配、排行榜、我的仓库、性格筛选、数据维护等功能。

## ✨ 主要功能

- 🔍 **宠物查询**：输入宠物名，显示其所有蛋组（14大蛋组）。
- 🧬 **种公匹配**：自动列出可孵蛋的其他宠物，按“可配种宠物总数”优先推荐优质种公，并标记已在仓库中的宠物及其性格。
- 🏆 **种公排行榜**：仅统计≥2蛋组的宠物，按可配种总数降序，支持分页。
- 📦 **种公仓库**（独立页面）：管理你拥有的宠物，支持按性格多选筛选，分析各蛋组覆盖情况，导入/导出备份。
- 🎭 **性格筛选器**：在仓库页面可按你拥有的性格多选筛选，动态显示蛋组覆盖及缺失提示。
- 🏠 **仓库匹配标记**：主页查询时，种公列表中会标明“已有”并显示该宠物在仓库中的所有性格。
- ⬆️ **返回顶部按钮**：长页面滚动时自动显示，点击平滑回顶部。
- 📝 **数据维护**：后台管理（密码`123`），支持增删行、CSV/JSON导入导出、重置初始数据。
- 💡 **性格九宫格**：添加宠物时使用一行五列的九宫格选择30种性格，并显示增减提示。
- 📱 **手机友好**：响应式设计，折叠卡片，触屏优化。

## 🚀 使用方法

### 在线使用（GitHub Pages）
👉 [https://cnrudy.github.io/Roco-Kingdom-StudMaster/](https://cnrudy.github.io/Roco-Kingdom-StudMaster/)

### 离线使用
1. 下载所有文件放入同一文件夹。
2. 双击 `index.html` 用浏览器打开（推荐 Chrome / Edge / 手机浏览器）。
3. 输入宠物名（支持联想）→ 查看蛋组及种公匹配 → 点击“添加到仓库”选择性格。
4. 点击“种公仓库”可管理已拥有宠物，使用性格筛选查看蛋组覆盖情况。
5. 需要修改原始蛋组数据时，进入 `admin.html` 输入密码 `123` 进行编辑。

## 🔧 数据维护说明

- 所有修改保存后，刷新主页即可生效。
- 支持导出 JSON 备份、导入 CSV 批量修改。
- 重置初始数据需输入“确认重置”防误触。
- **种公仓库数据**（您添加的宠物）可通过页面内的“导出仓库数据”备份。

## 🎭 性格相关

- 游戏内30种完整性格（聪明、固执、胆小等），添加宠物时九宫格可选，悬停显示增减效果。
- 仓库页面可按性格多选筛选，并提示当前性格下缺失哪些蛋组，方便针对性补充。

## ❓ 常见问题

**Q：为什么有些宠物查不到？**  
A：请确认输入的是**最小形态**名称（如“粉星仔”而非“星耀独角兽”）。本数据库统一使用最小形态。

**Q：种公排行榜里的“可配种宠物总数”是什么？**  
A：指该宠物所有蛋组下的**其他宠物去重总和**，总数越大代表能配种的宠物越多，是判断优质种公的核心指标。

**Q：如何修改后台密码？**  
A：用文本编辑器打开 `admin.html`，找到 `if (pwd === '123')`，将 `123` 替换为自己想要的密码。

**Q：我添加的宠物数据会丢失吗？**  
A：数据保存在浏览器的 `localStorage` 中，清除浏览器缓存或换设备会重置。建议定期在“种公仓库”页面点击“导出仓库数据”备份。

**Q：性格筛选器有什么用？**  
A：在仓库页面勾选你拥有的某些性格，系统会只显示这些性格的宠物，并分析它们覆盖了哪些蛋组、缺失哪些蛋组，帮你规划需要补充的种公。
