# 洛克王国世界 宠物蛋组种公查询器

纯前端、离线可用，支持种公匹配、排行榜、我的仓库、性格筛选等功能。

蛋组数据实时拉取自 [BWIKI 洛克王国 Wiki](https://wiki.biligame.com/rocom/)，无需手动更新。

## 功能

- 🔍 **宠物查询**：输入宠物名，显示其所有蛋组（14大蛋组）。
- 🧬 **种公匹配**：自动列出可孵蛋的其他宠物，按"可配种宠物总数"优先推荐优质种公，并标记已在仓库中的宠物及其性格。
- 🏆 **种公排行榜**：仅统计≥2蛋组的宠物，按可配种总数降序，支持分页。
- 📦 **种公仓库**（独立页面）：管理你拥有的宠物，支持按性格多选筛选，分析各蛋组覆盖情况，导入/导出备份。
- 🎭 **性格筛选器**：在仓库页面可按你拥有的性格多选筛选，动态显示蛋组覆盖及缺失提示。
- 🏠 **仓库匹配标记**：主页查询时，种公列表中会标明"已有"并显示该宠物在仓库中的所有性格。
- ⬆️ 返回顶部按钮。

## 数据来源

蛋组数据来自 [BWIKI 洛克王国 Wiki](https://wiki.biligame.com/rocom/) 的 `MediaWiki:Egg.json`，实时拉取，缓存 6 小时自动刷新。

## 文件

| 文件 | 作用 |
|------|------|
| `index.html` | 主查询页面（蛋组查询+种公匹配+添加到仓库） |
| `stud.html` | 种公排行榜（分页、排序） |
| `stud-repo.html` | 种公仓库（蛋组覆盖分析、性格筛选、导入导出） |
| `common.js` | 核心逻辑引擎（从BWIKI拉取数据、蛋组映射、种公匹配） |
| `style.css` | 统一样式 |

## 使用方法

👉 [在线访问](https://sxdl.github.io/Roco-Kingdom-StudMaster/)

- 下载所有文件放入同一文件夹。
- 双击 index.html 用浏览器打开（推荐 Chrome / Edge / 手机浏览器）。
- 或通过 GitHub Pages 直接访问。
- 输入宠物名（支持联想）→ 查看蛋组及种公匹配 → 点击"添加到仓库"选择性格。
- 种公仓库数据保存在浏览器 localStorage 中，可通过"导出仓库数据"备份。
