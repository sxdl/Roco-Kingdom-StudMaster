# 洛克王国 · 宠物蛋组查询器 & 优质种公推荐

> 一款纯前端、离线可用的洛克王国宠物蛋组查询工具，支持种公匹配、排行榜、数据维护等功能。

## ✨ 主要功能

- 🔍 **宠物查询**：输入宠物名，立即显示它所属的所有蛋组（14大蛋组）。
- 🧬 **种公匹配**：自动列出可与该宠物孵蛋的其他宠物（共同蛋组），并优先推荐“可配种宠物数”最多的优质种公。
- 🏆 **种公排行榜**：仅统计≥2蛋组的宠物，按“可配种宠物总数”降序排列，支持分页。
- 📝 **数据维护**：后台管理（密码保护），支持增删行、CSV/JSON 导入导出、重置初始数据。
- 🧩 **完全本地运行**：数据存储在浏览器中，无需网络，不收集任何信息。

## 🚀 使用方法

1. 将全部 6 个文件放入**同一文件夹**。
2. 双击 `index.html` 用浏览器打开（推荐 Chrome / Edge / 手机浏览器）。
3. 在搜索框输入宠物名（支持联想）→ 查看蛋组及种公匹配。
4. 点击蛋组标签可查看该蛋组所有宠物。
5. 点击种公名字可跳转查询该种公详情。
6. 需要修改数据时，进入 `admin.html` 输入密码 `123` 进行编辑。

## 🔧 数据维护说明

- 所有修改保存后，刷新 `index.html` 即可生效。
- 支持导出 JSON 备份、导入 CSV 批量修改。
- 内置“重置初始数据”功能（需输入“确认重置”防误触）。

## ❓ 常见问题

**Q：为什么有些宠物查不到？**  
A：请确认输入的是**最小形态**名称（如“粉星仔”而非“星耀独角兽”）。本数据库统一使用最小形态。

**Q：种公排行榜里的“可配种宠物总数”是什么？**  
A：指该宠物所有蛋组下的**其他宠物去重总和**，总数越大代表能配种的宠物越多，是判断优质种公的核心指标。

**Q：如何修改后台密码？**  
A：用文本编辑器打开 `admin.html`，找到 `if (pwd === '123')`，将 `123` 替换为自己想要的密码。

**Q：数据会丢失吗？**  
A：数据保存在浏览器的 `localStorage` 中，清除浏览器缓存或换设备会重置为初始数据。建议定期使用“导出JSON”备份。
